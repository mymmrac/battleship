# Battleship

Battleship is two players sea battle game

![Battleship](docs/screenshot-1.png)

## Install

Game + Server

```shell
go install github.com/mymmrac/battleship@latest
```

Server only

```shell
go install github.com/mymmrac/battleship/cmd/battleship-server@latest
```

## Start game server

```shell
battleship server
```

## Play

```shell
battleship
```
